# Wazuh SIEM Lab

## Overview

This project is a multi-platform SIEM lab that I'm building with Wazuh to simulate realistic security monitoring across Linux, Windows, and macOS.

This lab is designed to model how security teams collect data, detect suspicious activity, respond to incidents, find and remove vulnerabilities, and document decisions.

The current phase focuses on Linux/Ubuntu monitoring and hardening. Windows and macOS agents are planned for later phases.

## Project Goals

- Build a realistic Wazuh-based SIEM lab
- Monitor Linux, Windows, and macOS
- Collect meaningful data
- Detect suspicious activity
- Enhance alerts with threat intelligence
- Investigate malware, vulnerability, and configuration findings
- Remediate safely and verify results
- Document SOC-style investigation and response workflows
